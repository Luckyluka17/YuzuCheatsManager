Pourquoi Windows vous averti au lancement du logiciel ?

Au lancement du logiciel, il est très probable que Windows vous affiche cet avertissement :



Mais alors, pourquoi ? Et bien c'est simple, nous ne possédons pas de certificat pour désactiver cela. Notre logiciel n'est donc pas approuvé, ce qui signifie que par sécurité, Windows SmartScreen vous avertira au cas où le fichier téléchargé serait infecté par un virus. Pour vous montrer que notre logiciel ne contient pas de virus, vous pouvez analyser le code source ou vérifier sur Virus Total.
